# Python Server template

Template for a Python server using key libraries:

** FastAPI.
** SQLAlchemy.
** Pydantic
** JinJa2

The server integrate with a postgres database and has a basic CRUD API.

# Infrastructure

Contains example docker-compose setup with envoy proxy
